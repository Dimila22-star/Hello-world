# Hello JavaScript 

## Overview

This is a simple "Hello World" project in JavaScript. It demonstrates the basics of running JavaScript in the browser and using Node.js.

## Setup Instructions

### Browser

1. Open `index.html` in a web browser.
2. The message "Hello, World!" will appear in the console.

### Node.js

1. Install Node.js from [https://nodejs.org/](https://nodejs.org/).
2. Run the script:

```bash
node index.js
```

## Learning Reflection

* Learned how to structure a simple JS project.
* Practiced running JS in both browser and Node.js environments.
* Understood basic debugging using `console.log`.

## Common Errors

* ReferenceError: variable not defined → Ensure variables are declared.
* SyntaxError: Unexpected token → Check syntax carefully.
* TypeError → Avoid overwriting built-in objects like `console`.
